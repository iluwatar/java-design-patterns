## 或称
代理模式

## 目的
它是一种让对象将某种行为向外部表达，但实际上将实现该行为的责任委托给关联对象的技术。

## 类图
![alt text](./etc/delegation.png "Delegate")

## 适用性
使用委托模式以实现以下目的

* 降低类的耦合性
* 组件的行为相同，但是意识到这种情况将来可能会改变。
