## 目的
为了避免昂贵的资源重新获取，方法是在资源使用后不立即释放资源。资源保留其身份，保留在某些快速访问的存储中，并被重新使用，以避免再次获取它们。

## 适用性
在以下情况下使用缓存模式

* 重复获取，初始化和释放同一资源会导致不必要的性能开销。
